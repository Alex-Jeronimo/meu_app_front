# Organizador de Estudos - Front-end

## Descrição

Interface web do projeto **Organizador de Estudos**.

O front-end permite cadastrar, consultar, filtrar, editar, concluir e excluir atividades acadêmicas consumindo a API desenvolvida em Flask.

## Requisitos

- Navegador web atualizado.
- API do projeto em execução.
- Repositório `meu_app_api` configurado conforme o README da API.

## Instalação

Este front-end usa HTML, CSS e JavaScript puro. Por isso, não é necessário instalar dependências com `npm` ou outro gerenciador de pacotes.

Para configurar o ambiente local:

1. Baixe ou clone este repositório.
2. Deixe a API em execução.
3. Abra o arquivo `index.html` no navegador.

## Como executar

Primeiro, inicie a API seguindo as instruções do repositório `meu_app_api`.

Depois, abra o arquivo abaixo no navegador:

```text
meu_app_front/index.html
```

A API precisa continuar rodando enquanto o front-end estiver sendo usado.
