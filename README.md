# Organizador de Estudos - Front-end

Interface web do projeto **Organizador de Estudos**.

Ela permite cadastrar, consultar, filtrar, editar, concluir e excluir atividades acadêmicas consumindo a API desenvolvida em Flask.

## Como executar

Primeiro, deixe a API em execução. Para isso, utilize o README do repositório `meu_app_api`.

Depois, abra o arquivo no navegador:

```text
meu_app_front/index.html
```

A API precisa continuar rodando enquanto o front-end estiver sendo usado.
